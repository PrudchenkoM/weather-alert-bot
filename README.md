# 🌦 Weather Alert Bot

Python automation project that receives weather data from API and sends Telegram notifications.

## Features

- Get weather data from external API
- Parse JSON response
- Analyze temperature
- Send Telegram notification
- Error handling
- Logging system
- Environment variables for secrets

## Tech Stack

- Python
- Requests
- REST API
- JSON
- Telegram Bot API
- dotenv

## Project structure

```
weather-alert-bot

main.py
.env
.gitignore
requirements.txt
logs.txt
README.md
```

## How it works

Weather API
→ Python
→ JSON processing
→ Condition checking
→ Telegram notification
